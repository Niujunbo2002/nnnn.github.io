  5. MinerU 2.0: A End-to-End Vision-Language Model for High-Speed and High-Precision Document Parsing
    J Niu*, Z Liu*, Z Gu*, and 7 more authors
    Model Card
    Abs Blog Code Website

    MinerU 2.0 integrates a small-parameter, high-performance multimodal document parsing model for end-to-end high-speed and high-precision document understanding. With fewer than 1B parameters, it surpasses traditional 72B-level vision-language models in parsing accuracy, supports multilingual OCR, handwriting recognition, layout analysis, table parsing, formula recognition, and reading-order sorting, and reaches peak throughput above 10,000 tokens per second with sglang acceleration on a single NVIDIA 4090 GPU.
